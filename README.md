# Git and GitHub Workflow Mastery Project

This repository contains the deliverables for the Git & GitHub hands-on labs final project.

Included files:

- `LICENSE` — Apache 2.0 license
- `README.md` — this file
- `CODE_OF_CONDUCT.md` — contributor code of conduct
- `CONTRIBUTING.md` — contribution guidelines
- `simple-interest.sh` — shell script that calculates simple interest

Usage:

1. Make the script executable: `chmod +x simple-interest.sh`
2. Run: `./simple-interest.sh <principal> <rate_percent> <years>`

Example:

```
./simple-interest.sh 1000 5 3
# Output: Simple interest for principal 1000 at 5% over 3 years is: 150
```
